# 公众号配图指南

## 封面图设计原则

### 尺寸规范
- 头条封面：900 x 383 px（2.35:1）
- 次条封面：500 x 500 px（1:1）
- 正文配图：宽度 900px，高度自适应

### 设计要点
1. **文字简洁**：不超过 10 个字，字体要大
2. **色彩对比**：背景与文字形成强烈对比
3. **情绪匹配**：科技感用蓝/黑，温暖感用橙/黄
4. **留白适当**：不要塞满，呼吸感很重要

## 配图类型推荐

### 技术类文章
- **代码截图**：IDE 暗色主题，高亮关键行
- **架构图**：使用 Draw.io、Excalidraw 风格
- **流程图**：简洁明了，避免过于复杂
- **终端截图**：展示命令执行效果
- **性能图表**：数据可视化，趋势明显

### 产品/工具类
- **界面截图**：完整界面 + 局部放大
- **对比图**：Before/After 左右对比
- **功能演示**：GIF 动图展示交互
- **实物照片**：如果是硬件产品

### 观点/故事类
- **场景照片**：与内容相关的真实场景
- **人物照片**：增加真实感和代入感
- **聊天记录**：截图展示真实对话
- **手绘插图**：增加趣味性和独特性

## 配图注意事项

1. **版权问题**：优先使用免费商用素材或自己拍摄
2. **文件大小**：单张图片不超过 2MB
3. **清晰度**：避免模糊或压缩过度的图片
4. **一致性**：全文配图风格保持一致
5. **相关性**：图片必须与内容相关，不要为了配图而配图
